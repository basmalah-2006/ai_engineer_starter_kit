# AI Engineer Starter Kit

Week 1 project from the 12-Week AI Industry Roadmap by Helwan Career Center.

## Description
This project combines a REST API call with Hugging Face's sentiment analysis pipeline. It fetches a random quote from a public API and analyzes the sentiment of 10 sentences using a pre-trained transformer model.

## Features
- Calls a public REST API with timeout and error handling
- Uses Hugging Face `transformers` pipeline for sentiment analysis
- Fully reproducible environment via `requirements.txt`
- Clean Git history with meaningful commits
